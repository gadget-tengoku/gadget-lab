# ガジェット天国 - スコアリングロジック完全公開

> [ガジェット天国](https://gadget-tengoku.com) の商品ランキングを算出している実コードを公開しているリポジトリです。

## なぜ公開するのか

当サイトはアフィリエイトメディアです。商品ランキングを掲載していますが、「広告主に忖度していないか」「順位の根拠は何か」を読者が検証できなければ、ランキングは信用できません。

そこでスコア算出の**全数式・全パラメータ・全除外ルール**をGitHubで公開しています。

ランキング = `scoring.py` を実行した結果。それ以上でも以下でもありません。

---

## 当サイトの立ち位置

**当サイトは実機を所有していません。** マイベスト・価格.comマガジン・Rentioのような実機検証メディアとは別の戦い方をしています。

代わりに：

| やっていること | やっていないこと |
|---|---|
| 楽天市場の実売データを機械的に加工 | 実機での吸引力測定・音質評価・装着感 |
| レビュー件数×評価点で重み付け | 長期使用での耐久性検証 |
| 消耗品・部品系の商品を除外 | メーカーサポート品質の評価 |
| 商品名から特徴キーワードを抽出 | 楽天以外の販売チャネル分析 |

**これは弱みです。隠しません。** 購入前にマイベスト・価格.comマガジン等の実機検証記事も併せてご確認ください。

## スコア算出式（100点満点）

```
total_score =
    review_score      [上限35点]
  + cost_score        [10〜25点]
  + brand_score       [5〜10点]
  + feature_bonus     [上限15点]
  + baseline          [10点]
```

### 1. レビュースコア（上限35点）

```python
review_score = review_avg × log10(review_count + 1) × 5
```

ポイントは **log10（対数）でレビュー件数を効かせる** こと。

| 商品 | 評価 | 件数 | 計算 | スコア |
|---|---|---|---|---|
| 商品A | ★4.5 | 50件 | 4.5 × log10(51) × 5 | 38.5 → **35.0**（上限） |
| 商品B | ★4.4 | 900件 | 4.4 × log10(901) × 5 | 64.9 → **35.0**（上限） |
| 商品C | ★4.7 | 30件 | 4.7 × log10(31) × 5 | 33.9 |
| 商品D | ★0.0 | 0件 | — | **None（除外）** |

「★4.7だけど30件」より「★4.4で900件」を上に評価したい、というのが設計意図。

### 2. 価格帯マッチング（10〜25点）

カテゴリの「中位価格帯」に入る商品を最大評価。

```python
mid_lo = min_price × 1.5
mid_hi = max_price × 0.7

if mid_lo <= price <= mid_hi:     cost_score = 25  # 中位価格帯
elif min_price <= price <= max_price: cost_score = 18  # 範囲内
else:                                  cost_score = 10  # 範囲外
```

極端な高額品・激安品より、買いやすい価格帯の商品を優先します。

### 3. ブランド信頼度（5〜10点）

商品名に含まれるブランド名から重み付けスコアを取得。

[ブランドリストの完全版を見る](./scoring.py) （`BRAND_TRUST` 辞書）

| ブランド | スコア |
|---|---|
| Sony, Apple | 10 |
| Bose, Jabra, Logicool, Garmin, Panasonic, Shure | 9 |
| Anker, Soundcore, Razer, SteelSeries, BenQ, ASUS, LG | 8 |
| Xiaomi, Fitbit, CIO, エレコム など | 7 |
| 無名・不明 | 5 |

評価基準：楽天市場での販売実績・ユーザー満足度・サポート品質を編集部が独自評価。

### 4. 特徴キーワードボーナス（上限15点）

カテゴリで重要な機能キーワードが商品名に含まれていればスコア加算。

例：イヤホンカテゴリでは以下のキーワード（カテゴリ設定 `features`）：

- ノイズキャンセリング / ANC
- ハイレゾ / LDAC
- マルチポイント
- ワイヤレス / Bluetooth

それぞれ含まれていれば +3点（上限15点）。

### 5. ベースライン

固定で +10点。極端なスコア差を緩和する役割。

## 商品除外ルール

以下のいずれかに該当する商品は、スコア計算前に**候補から完全に除外**されます：

### 完全除外（必須）

- ✗ 評価点 = 0.0
- ✗ レビュー件数 = 0
- ✗ 価格 ≤ 0

### カテゴリ別除外（フィルタ）

各カテゴリで以下を設定：

- **min_price / max_price**：価格帯のレンジ外
- **min_review_count**：レビュー件数の下限未満（フォールバック緩和あり）
- **min_review_average**：評価点の下限未満
- **min_title_length**：商品名が短すぎる（型番のみなど）
- **exclude キーワード**：消耗品・部品系

### exclude キーワード例（イヤホンの場合）

```python
"exclude": [
    "イヤーピース", "イヤーチップ", "ケース単体", "充電ケーブル",
    "交換用", "互換品", "保護フィルム", "ストラップ", "シール"
],
```

これにより、楽天検索結果に紛れ込む「Sony WF-1000XM5 用 交換イヤーピース ¥800」のような消耗品が、本体商品のランキングに混入することを防いでいます。
