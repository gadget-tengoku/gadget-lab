DEVICE TROUBLE · KEYBOARD

キーボードを押しても反応しないときの確認順｜Windows PC

公開・更新：2026年6月27日　／　執筆：ガジェット天国 管理人

Windowsでキーボードが反応しないときは、故障と決めつける前に「キーボード本体」「接続」「Windows」のどこに原因があるかを順番に切り分けます。

先に結論別USBポート、PC再起動、USBハブを外す、別PCで試す、の4つで本体故障かPC側かを早く分けられます。
確認する順番
  1. 基本確認
  2. 有線
  3. 無線
  4. Windows
  5. 修理判断

1. 最初の4項目

  1. PCを再起動する
  2. 別のUSBポートへ挿す
  3. USBハブを外して直接接続する
  4. 別のPCや端末で反応するか試す

2. 有線キーボード

  • 着脱式ケーブルなら両端を挿し直す
  • 別の対応ケーブルで比較
  • ケーブルの折れや端子のぐらつきを確認
  • 背面USBポートでも試す

3. 無線・Bluetoothキーボード

確認項目
  • 電源スイッチと充電・電池残量
  • 2.4GHzレシーバーの挿し直し
  • Bluetoothの削除と再ペアリング
  • 接続モードの切り替え
  • 他端末へ自動接続されていないか

4. Windows側を確認する

  • デバイスマネージャーに警告がないか
  • フィルターキーなどアクセシビリティ設定
  • メーカーソフトを終了して比較
  • Windows Update後なら再起動
  • 特定アプリだけならアプリ側のショートカット設定

症状で判断する

症状考えられる場所
LEDも点かない電源、ケーブル、USBポート、本体
一部キーだけ反応しないキー割り当て、接点、スイッチ
ログイン画面では動く常駐ソフトやアプリ設定
別PCでも動かない本体故障の可能性が高い

5. 修理・交換を考える目安

複数のPC、複数のポート、別ケーブルでも反応しない場合は、本体側の可能性が高くなります。購入日、保証期間、症状の動画を準備して販売店やメーカーへ相談します。
