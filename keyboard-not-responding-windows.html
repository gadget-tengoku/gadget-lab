キーボードを押しても反応しないときの確認順｜Windows PC向け

公開日：2026年6月27日　／　執筆：ガジェット天国 管理人

キーボードが急に反応しなくなったときは、故障と決めつける前に「本体」「接続」「Windows」のどこに原因があるかを切り分けます。

この記事の方針
確認できていない性能や使用感は断定せず、原因を切り分ける順番と購入前の確認項目を中心にまとめています。

1. 有線・無線共通の確認

  • PCを再起動する
  • 別のUSBポートへ接続する
  • USBハブを外してPCへ直接接続する
  • 別のPCや端末で反応するか確認する

2. 無線キーボード

確認項目
  • 電源スイッチ
  • 充電・電池残量
  • 2.4GHzレシーバーの挿し直し
  • Bluetoothの再ペアリング
  • 接続モードの切り替え

3. Windows側

  • デバイスマネージャーで警告がないか確認
  • Windows Update直後なら再起動
  • フィルターキーなどアクセシビリティ設定を確認
  • メーカーソフトを一度終了して比較

4. 交換・修理を考える目安

複数のPCで反応せず、ケーブルやレシーバーを変えても改善しない場合は、本体故障の可能性が高くなります。保証期間と購入履歴を確認します。
