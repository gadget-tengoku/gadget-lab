【GitHubへ反映する方法】
一番簡単なのは、このフォルダの中身をGitHubリポジトリへ上書きアップロードする方法です。
ただしファイル数が多いため、GitHub Desktopを使う方が安全です。

1. GitHub Desktopをインストール
2. gadget-labをClone
3. Cloneしたフォルダへ、この修正版の中身をすべて上書きコピー
4. GitHub DesktopでSummaryに「サイト再構築」と入力
5. Commit to main → Push origin
